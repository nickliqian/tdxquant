| 值 | 含义 |
|------|------|
| none | 不复权 |
| front | 前复权 |
| back | 后复权 |

| 值 | 含义 |
|------|------|
| 0 | 不复权 |
| 1 | 前复权 |
| 2 | 后复权 |
